# CheckpointPro
#### A dead-simple Version Control Software made for education

CheckpointPro is a slimmed-down alternative to Git for the classroom environment. CheckpointPro offers an easy and intuitive interface to make Version Control approachable.
Students can save their progress as they work and roll back to earlier versions if something breaks.

<img width="521" height="413" alt="image" src="https://github.com/user-attachments/assets/ba26ff30-bf10-47fb-a83f-62315b04691b" />

## Features

### Create Checkpoints
Save progress as you go, establishing a workflow for labeling and documenting your work. Timestamp every change to track timelines.
Rewind the project to old versions to review progress.

### Project Reports
Generate Project Reports that summarize how a project progressed over time. Document timelines, and spot suspiciously large additions.

<img width="433" height="415" alt="image" src="https://github.com/user-attachments/assets/b6555707-90b2-4162-87b4-07251a06b0cc" />

### Review Unsaved Work
Students can review their unsaved changes to see what they changed before something broke.

<img width="667" height="518" alt="image" src="https://github.com/user-attachments/assets/5c8f3406-863a-4ce1-a178-b30917fd42ca" />

### Share with students
Students can install and run CheckpointPro for themselves on their own computers and laptops.
